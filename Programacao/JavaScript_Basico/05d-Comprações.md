# Comparações

Servem para comparar dois valores e/ou variáveis.

- Igual - `==`
- Diferente - `!=`
- Menor - `<`
- Maior - `>`
- Menor igual - `<=`
- Maior igual - `>=`

Obs : Sempre retornam verdadeiro (true) ou false (false)

```javascript
1 > 2
1 == 1
"A" < "B"
```

Para Variáveis

```javascript
const idade =16;
const idade = 18;

idade >= 18;
```
